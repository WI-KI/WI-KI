[http://mohu.org/info/symbols/symbols.htm](http://mohu.org/info/symbols/symbols.htm)

行内公式：

```markdown
$e^x$
```

块级公式：

```markdown
$$
e^x
$$
```

请不要使用以下语法：

```markdown
$$e^x$$
```
